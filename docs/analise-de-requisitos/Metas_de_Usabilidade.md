# Metas de Usabilidade

## Introdução

Este artefato tem como propósito definir as metas de usabilidade que estabelecem quais fatores de qualidade de uso devem ser priorizados no projeto, além de como a avaliação ocorrerá
durante o processo de design quais faixas de valores são aceitáveis ou inaceitáveis. As mestas de usabilidades tratadas nesse documento são as definidas pelos pesquisador Jacob Nielsen,
sendo fundamentais para o design de sistemas computacionais  mais intuitivos e agradáveis para os usuários dentro da nossa aplicação.
